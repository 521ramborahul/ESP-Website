Scheduling {{ prog.niceName }}

This is the page for assigning times and rooms to classes for {{ prog.niceName }}. Please try refreshing everything if the schedules are not updating correctly. This can take a long time!

There are a few scheduling rules obeyed by this interface; you can override them using the individual "Manage Class" pages if necessary. You must assign a time block to a class before you can assign a classroom or resources. If you change the time block for a class, its room and resource assignments will be deleted. This is to ensure that you don't create a conflict by moving the class to an infeasible time.

The Security Schedule page lists all your current class schedules, by time block.
